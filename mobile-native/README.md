# CareerBoost AI - Mobile App

Native Android app built with React Native and Expo.

## Features
- AI Resume Builder
- ATS Score Checker
- Cover Letter Generator
- Payment Integration
- Offline Support

## Tech Stack
- React Native
- Expo SDK 50
- WebView for web content
- Native Android components

## Build
```bash
npm install
eas build --platform android --profile preview
```

## Version
1.0.0
